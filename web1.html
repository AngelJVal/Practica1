1990-2000 Nacimiento de la Web 1.0

1990 – Nace la World Wide Web

En 1990, Tim Berners-Lee desarrolló la World Wide Web, creando las principales tecnologías que permitirían el funcionamiento de la web: el lenguaje HTML para la estructuración de los documentos, el protocolo HTTP para la transferencia de información, así como el primer servidor web y el primer navegador web, conocido como WorldWideWeb, sentando así las bases de la navegación en Internet.

1991 – Primer sitio web

Se publica el primer sitio web, explicando el funcionamiento de la WWW.

1994 – Fundación del W3C

Se crea el World Wide Web Consortium (W3C) para estandarizar las tecnologías web.

1995 – Aparición de JavaScript

Se introduce JavaScript, permitiendo páginas web dinámicas.

1995 – Nace el comercio electrónico

Aparecen Amazon y eBay, impulsando las compras en línea.

1996 – Introducción de CSS

Se publican las Hojas de Estilo en Cascada (CSS), separando diseńo y contenido.

2004 – Transición hacia Web 2.0

Se populariza el término Web 2.0, marcando el fin de la Web 1.0.
